## Upgrade adm-zip to version 0.4.11 or later

"dependencies": {
  "adm-zip": ">=0.4.11"
}
or…
"devDependencies": {
  "adm-zip": ">=0.4.11"
}

## https-proxy-agent

Remediation
Upgrade https-proxy-agent to version 2.2.0 or later. For example:

"dependencies": {
  "https-proxy-agent": ">=2.2.0"
}
or…
"devDependencies": {
  "https-proxy-agent": ">=2.2.0"
}
